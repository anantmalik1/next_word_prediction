## 🌌 <span style="color:#00FFF7">Project Overview</span>

> ⚡ This project uses a **Deep Learning LSTM Model** to predict the next word in a sentence.
> 🧠 It understands patterns in text and generates smart predictions like a mini AI assistant.

---

## ✨ <span style="color:#FF00FF">Features</span>

* ⚡ Real-time next word prediction
* 🧠 LSTM Neural Network
* 📊 NLP-based training
* 🚀 Fast inference
* 🎯 Beginner-friendly AI project
